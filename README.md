# Volunteer Registration and Management System

## Overview

The Volunteer Registration and Management System is a frontend web application developed using HTML, CSS, and JavaScript. It allows users to register volunteers, store their details in browser Local Storage, and manage records dynamically through an interactive table interface.

This project demonstrates practical implementation of DOM manipulation, event handling, form validation, and client-side data persistence.
---

## Features

- Volunteer registration form  
- Custom JavaScript validation  
- Data storage using Browser Local Storage  
- Dynamic table rendering  
- Search functionality  
- Availability-based active status  
- Form reset after submission  

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Browser Local Storage**

---

## How It Works

1. User fills out the volunteer registration form.
2. Input validation is performed using JavaScript.
3. Data is stored in Local Storage as structured objects.
4. Stored records are retrieved and displayed in a dynamic table.
5. Search functionality filters records.
6. Status is managed based on selected availability time.

---

## How to Run the Project

1. Clone the repository:
2. Open the project folder.
3. Open `index.html` in your browser.

No backend setup is required.
